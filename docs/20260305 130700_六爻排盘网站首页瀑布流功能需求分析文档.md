## 一、项目概述

### 1.1 项目背景

六爻排盘网站是一个为用户提供专业六爻占卜排盘服务的平台。当前项目已完成基础排盘功能、用户系统、排盘记录管理等核心模块。为提升用户体验和内容展示效果，计划开发仿小红书风格的首页瀑布流展示功能。

### 1.2 项目目标

开发一个类似小红书 `https://www.xiaohongshu.com/explore` 的首页瀑布流展示功能，实现以下核心目标：

1. **提升用户体验**：通过瀑布流布局，使排盘结果展示更加美观、直观
2. **增加用户互动**：实现点赞、收藏等互动功能，提升用户参与度
3. **优化内容发现**：通过瀑布流和无限滚动，使用户更方便地发现感兴趣的排盘内容
4. **响应式设计**：适配不同设备屏幕尺寸，提供一致的用户体验

### 1.3 参考标准

参考小红书PC端瀑布流展示效果：
- 纯白背景（#FFFFFF）
- 卡片圆角12px
- 列间距15px，行间距15px
- 卡片三层结构：图片区、标题区、用户互动区
- 导航栏悬浮吸顶，滚动时背景变化
