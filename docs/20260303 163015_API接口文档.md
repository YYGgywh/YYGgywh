## 一、API概述

### 1.1 基础信息

- **API基础路径**：`/api`
- **请求方法**：GET、POST、PUT、DELETE
- **响应格式**：JSON
- **数据格式**：所有时间字段使用Unix时间戳（秒）
- **编码格式**：UTF-8

### 1.2 响应结构

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

- **code**：状态码，200表示成功，其他表示错误
- **msg**：响应消息
- **data**：响应数据

---

## 二、认证机制

### 2.1 JWT Token

- **认证方式**：Bearer Token
- **请求头**：`Authorization: Bearer <token>`
- **Token有效期**：24小时
- **刷新机制**：暂不支持自动刷新

### 2.2 权限级别

- **普通用户**：role=0
- **管理员**：role=1
- **超级管理员**：role=99

## 三、用户管理API

### 3.1 发送验证码

- **接口**：`POST /api/user/send_code`
- **功能**：发送手机验证码
- **请求参数**：
  | 参数 | 类型 | 必选 | 描述 |
  |------|------|------|------|
  | phone | string | 是 | 手机号，格式：1[3-9]\d{9} |
- **响应**：
  ```json
  {
    "code": 200,
    "msg": "验证码发送成功",
    "data": null
  }
  ```

### 3.2 发送邮箱验证码

- **接口**：`POST /api/user/send_email_code`
- **功能**：发送邮箱验证码
- **请求参数**：
  | 参数 | 类型 | 必选 | 描述 |
  |------|------|------|------|
  | email | string | 是 | 邮箱地址 |


### 3.3 用户注册

- **接口**：`POST /api/user/register`
- **功能**：用户注册（支持手机或邮箱）
- **请求参数**：
  | 参数 | 类型 | 必选 | 描述 |
  |------|------|------|------|
  | phone | string | 否 | 手机号 |
  | email | string | 否 | 邮箱地址 |
  | login_name | string | 否 | 登录名，4-20位 |
  | code | string | 是 | 验证码，6位数字 |
  | password | string | 是 | 密码，至少6位 |
- **响应**：
  ```json
  {
    "code": 200,
    "msg": "注册成功",

  ```

## 七、错误码说明

| 错误码 | 描述 | 说明 |
|--------|------|------|
| 400 | 参数错误 | 请求参数格式不正确或缺少必要参数 |
| 401 | 未授权 | 未提供有效的认证信息或Token无效 |
| 403 | 权限不足 | 当前用户无权限执行此操作 |
| 404 | 资源不存在 | 请求的资源不存在 |
| 500 | 服务器内部错误 | 服务器处理请求时发生错误 |

## 八、使用示例

### 8.1 Python请求示例

```python
import requests

# 登录获取Token
def login():
    url = "http://localhost:8000/api/user/login"
    data = {
        "phone": "13812345678",
        "password": "123456"
    }
    response = requests.post(url, json=data)
    return response.json()

# 获取排盘记录
def get_pan_records(token):
    url = "http://localhost:8000/api/pan/list"
    headers = {
        "Authorization": f"Bearer {token}"
    }
    response = requests.get(url, headers=headers)
    return response.json()

# 保存排盘记录
def save_pan_record(token, pan_params, pan_result):
    url = "http://localhost:8000/api/pan/save"
    headers = {
        "Authorization": f"Bearer {token}"
    }
    data = {
        "pan_type": "liuyao",
        "pan_params": pan_params,
        "pan_result": pan_result
    }
    response = requests.post(url, json=data, headers=headers)
    return response.json()
```

### 8.2 JavaScript请求示例

```javascript
// 登录获取Token
async function login() {
  const response = await fetch('http://localhost:8000/api/user/login', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      phone: '13812345678',
      password: '123456'
    })
  });
  return await response.json();
}

// 获取排盘记录
async function getPanRecords(token) {
  const response = await fetch('http://localhost:8000/api/pan/list', {
    headers: {
      'Authorization': `Bearer ${token}`
    }
  });
  return await response.json();
}

// 保存排盘记录
async function savePanRecord(token, panParams, panResult) {
  const response = await fetch('http://localhost:8000/api/pan/save', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${token}`
    },
    body: JSON.stringify({
      pan_type: 'liuyao',
      pan_params: panParams,
      pan_result: panResult
    })
  });
  return await response.json();
}
```
