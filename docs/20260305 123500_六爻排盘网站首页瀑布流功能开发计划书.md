## 一、项目背景分析

### 1.1 项目概述

本项目为六爻排盘网站，旨在为用户提供专业的六爻占卜排盘服务。当前项目已完成基础排盘功能、用户系统、排盘记录管理等核心模块。为提升用户体验和内容展示效果，计划开发仿小红书风格的首页瀑布流展示功能。

### 1.2 现有技术栈

**前端技术栈：**
- React 19.2.3 - 核心UI框架
- React Router v7 - 路由管理
- Axios 1.13.5 - HTTP请求库
- CSS3/Flexbox/Grid - 样式布局

**后端技术栈：**
- FastAPI 0.133.0 - Web框架
- SQLAlchemy 2.0.31 - ORM框架
- SQLite - 数据库
- Python 3.13 - 开发语言

**现有数据库表结构：**
- `pan_record` - 排盘记录表
- `user` - 用户表
- `comment` - 评论表

### 1.3 开发目标

基于小红书 `https://www.xiaohongshu.com/explore` 的展示风格，实现以下核心功能：

1. **无规则瀑布流布局** - PC端3-5列自适应，移动端单列
2. **响应式设计** - 适配1920×1080、1440×900、1280×960等主流分辨率
3. **无限滚动加载** - 初始加载12张卡片，滚动触底加载6张
4. **图片懒加载** - 提升首屏加载性能
5. **卡片交互** - Hover效果、点击弹窗详情页
6. **数据缓存策略** - 优化重复数据请求

### 1.4 参考标准

参考小红书PC端瀑布流展示效果：
- 纯白背景（#FFFFFF）
- 卡片圆角12px
- 列间距15px，行间距15px
- 卡片三层结构：图片区、标题区、用户互动区
- 导航栏悬浮吸顶，滚动时背景变化

#### 3.2.2 状态管理

使用React Context进行状态管理：

```javascript
// contexts/WaterfallContext.jsx
const WaterfallContext = createContext();

const WaterfallProvider = ({ children }) => {
  const [cards, setCards] = useState([]);
  const [loading, setLoading] = useState(false);
  const [hasMore, setHasMore] = useState(true);
  const [selectedCard, setSelectedCard] = useState(null);
  const [isModalOpen, setIsModalOpen] = useState(false);
  
  // 加载更多数据
  const loadMore = async () => {
    if (loading || !hasMore) return;
    setLoading(true);
    try {
      const newCards = await fetchPanRecords(page, size);
      setCards(prev => [...prev, ...newCards]);
      setHasMore(newCards.length === size);
    } catch (error) {
      console.error('加载失败:', error);
    } finally {
      setLoading(false);
    }
  };
  
  // 打开弹窗
  const openModal = (card) => {
    setSelectedCard(card);
    setIsModalOpen(true);
  };
  
  // 关闭弹窗
  const closeModal = () => {
    setIsModalOpen(false);
    setSelectedCard(null);
  };
  
  return (
    <WaterfallContext.Provider value={{
      cards, loading, hasMore, selectedCard, isModalOpen,
      loadMore, openModal, closeModal
    }}>
      {children}
    </WaterfallContext.Provider>
  );
};
```

### 5.1 瀑布流布局实现

#### 5.1.1 核心算法

```javascript
// utils/waterfallLayout.js

/**
 * 计算瀑布流布局
 * @param {Array} items - 卡片数据数组
 * @param {number} containerWidth - 容器宽度
 * @param {number} columnCount - 列数
 * @param {number} gap - 间距
 * @returns {Array} 布局信息数组
 */
export const calculateWaterfallLayout = (items, containerWidth, columnCount, gap = 15) => {
  // 计算每列宽度
  const columnWidth = (containerWidth - (columnCount - 1) * gap) / columnCount;
  
  // 初始化每列高度
  const columnHeights = new Array(columnCount).fill(0);
  
  // 计算每个卡片的位置
  const layouts = items.map((item, index) => {
    // 找到高度最小的列
    const minHeightIndex = columnHeights.indexOf(Math.min(...columnHeights));
    
    // 计算卡片位置
    const left = minHeightIndex * (columnWidth + gap);
    const top = columnHeights[minHeightIndex];
    
    // 估算卡片高度（根据图片高度+文字区域高度）
    const itemHeight = item.imageHeight || 400; // 默认400px
    const textHeight = item.title ? 60 : 40; // 标题区域高度
    const userHeight = 40; // 用户信息区域高度
    const totalHeight = itemHeight + textHeight + userHeight + gap;
    
    // 更新列高度
    columnHeights[minHeightIndex] += totalHeight;
    
    return {
      index,
      left,
      top,
      width: columnWidth,
      height: totalHeight - gap,
      columnIndex: minHeightIndex
    };
  });
  
  return {
    layouts,
    containerHeight: Math.max(...columnHeights)
  };
};

/**
 * 根据屏幕宽度获取列数
 * @param {number} width - 屏幕宽度
 * @returns {number} 列数
 */
export const getColumnCount = (width) => {
  if (width >= 1200) return 3;
  if (width >= 992) return 2;
  return 1;
};
```

#### 5.1.2 瀑布流容器组件

```javascript
// components/Waterfall/WaterfallContainer.jsx

import React, { useState, useEffect, useRef, useCallback } from 'react';
import { calculateWaterfallLayout, getColumnCount } from '../../utils/waterfallLayout';
import { throttle } from '../../utils/throttle';
import WaterfallCard from './WaterfallCard';
import LoadingDots from '../Common/LoadingDots';
import EmptyState from '../Common/EmptyState';
import './WaterfallContainer.css';

const WaterfallContainer = ({ 
  items, 
  loading, 
  hasMore, 
  onLoadMore, 
  onCardClick,
  onLikeToggle 
}) => {
  const containerRef = useRef(null);
  const [layouts, setLayouts] = useState([]);
  const [containerHeight, setContainerHeight] = useState(0);
  const [columnCount, setColumnCount] = useState(3);

  // 计算布局
  const updateLayout = useCallback(() => {
    if (!containerRef.current || items.length === 0) return;
    
    const containerWidth = containerRef.current.offsetWidth;
    const newColumnCount = getColumnCount(containerWidth);
    setColumnCount(newColumnCount);
    
    const { layouts: newLayouts, containerHeight: newHeight } = calculateWaterfallLayout(
      items,
      containerWidth,
      newColumnCount,
      15 // gap
    );
    
    setLayouts(newLayouts);
    setContainerHeight(newHeight);
  }, [items]);

  // 窗口大小变化时重新计算布局
  useEffect(() => {
    const handleResize = throttle(() => {
      updateLayout();
    }, 200);

    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, [updateLayout]);

  // 数据变化时重新计算布局
  useEffect(() => {
    updateLayout();
  }, [items, updateLayout]);

  // 无限滚动
  useEffect(() => {
    const handleScroll = throttle(() => {
      const scrollTop = window.pageYOffset || document.documentElement.scrollTop;
      const windowHeight = window.innerHeight;
      const documentHeight = document.documentElement.scrollHeight;
      
      // 距离底部500px时触发加载
      if (scrollTop + windowHeight >= documentHeight - 500) {
        onLoadMore();
      }
    }, 300);

    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, [onLoadMore]);

  if (items.length === 0 && !loading) {
    return <EmptyState />;
  }

  return (
    <div 
      ref={containerRef}
      className="waterfall-container"
      style={{ height: containerHeight }}
    >
      {items.map((item, index) => {
        const layout = layouts[index];
        if (!layout) return null;
        
        return (
          <div
            key={item.id}
            className="waterfall-item"
            style={{
              position: 'absolute',
              left: layout.left,
              top: layout.top,
              width: layout.width,
              opacity: layout ? 1 : 0,
              transition: 'opacity 0.3s ease'
            }}
          >
            <WaterfallCard
              data={item}
              onClick={() => onCardClick(item)}
              onLikeToggle={(e) => {
                e.stopPropagation();
                onLikeToggle(item);
              }}
            />
          </div>
        );
      })}
      
      {loading && (
        <div className="waterfall-loading">
          <LoadingDots />
        </div>
      )}
      
      {!hasMore && items.length > 0 && (
        <div className="waterfall-no-more">已加载全部内容</div>
      )}
    </div>
  );
};

export default WaterfallContainer;

      setPage(2);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  }, []);

  // 加载更多
  const loadMore = useCallback(async () => {
    if (loading || !hasMore) return;
    
    setLoading(true);
    setError(null);
    try {
      const response = await getPanList(page, LOAD_MORE_SIZE);
      if (response.data.length > 0) {
        setCards(prev => [...prev, ...response.data]);
        setPage(prev => prev + 1);
      }
      setHasMore(response.data.length === LOAD_MORE_SIZE);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  }, [page, loading, hasMore]);

  // 重试加载
  const retry = useCallback(() => {
    if (cards.length === 0) {
      loadInitial();
    } else {
      loadMore();
    }
  }, [cards.length, loadInitial, loadMore]);

  useEffect(() => {
    loadInitial();
  }, [loadInitial]);

  return {
    cards,
    loading,
    hasMore,
    error,
    loadMore,
    retry
  };
};
```

// hooks/useInfiniteScroll.js

import { useEffect, useCallback } from 'react';
import { throttle } from '../utils/throttle';

const DEFAULT_THRESHOLD = 500;

export const useInfiniteScroll = (callback, threshold = DEFAULT_THRESHOLD) => {
  const handleScroll = useCallback(
    throttle(() => {
      const scrollTop = window.pageYOffset || document.documentElement.scrollTop;
      const windowHeight = window.innerHeight;
      const documentHeight = document.documentElement.scrollHeight;
      
      if (scrollTop + windowHeight >= documentHeight - threshold) {
        callback();
      }
    }, 300),
    [callback, threshold]
  );

  useEffect(() => {
    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, [handleScroll]);
};

@router.post("/like/toggle", response_model=LikeToggleResponse)
async def toggle_like(
    request: LikeToggleRequest,
    current_user: User = Depends(get_current_user_from_payload),
    db: Session = Depends(get_db),
    req: Request = Depends(rate_limit_dependency)
):
    """
    点赞/取消点赞排盘记录
    """
    # 查询排盘记录
    pan_record = db.query(PanRecord).filter(
        PanRecord.id == request.pan_id,
        PanRecord.deleted_at.is_(None)
    ).first()
    
    if not pan_record:
        raise HTTPException(status_code=404, detail="排盘记录不存在")
    
    # 查询是否已点赞
    existing_like = db.query(PanLike).filter(
        PanLike.user_id == current_user.id,
        PanLike.pan_id == request.pan_id
    ).first()
    
    if existing_like:
        # 取消点赞
        db.delete(existing_like)
        pan_record.like_count = max(0, pan_record.like_count - 1)
        is_liked = False
    else:
        # 添加点赞
        new_like = PanLike(
            user_id=current_user.id,
            pan_id=request.pan_id
        )
        db.add(new_like)
        pan_record.like_count += 1
        is_liked = True
    
    db.commit()
    
    return LikeToggleResponse(
        data={
            "is_liked": is_liked,
            "like_count": pan_record.like_count
        }
    )
