## 一、项目概述

### 1.1 项目背景

本项目为六爻排盘结果渲染网站，采用小红书式瀑布流布局展示用户发布的排盘记录。本次开发任务为实现瀑布流卡片点击弹窗功能，用户点击任意排盘卡片后弹出详情弹窗，展示完整的排盘结果和专业解读信息，同时支持用户互动功能（点赞、收藏、评论、关注等）。

### 1.2 开发目标

1. 实现符合设计规范的卡片详情弹窗
2. 展示完整的六爻排盘结果（卦象、四柱、六爻详细信息）
3. 实现用户互动功能（点赞、收藏、评论、关注）
4. 支持响应式布局，适配PC端和移动端
5. 确保高性能渲染和流畅的用户体验

### 1.3 需求分析

**功能需求**：

- 弹窗蒙层覆盖全屏，点击可关闭
- 弹窗主体采用左右分栏布局（PC端），上下布局（移动端）
- 左侧展示六爻排盘结果，包含四柱信息、卦宫卦名、六爻详细信息、世身信息等
- 右侧展示用户信息和互动区域，包含用户头像、昵称、关注按钮、标题、标签、评论区等
- 支持点赞、收藏、评论、关注等互动功能
- 评论输入框点击展开，输入评论内容并发布

**非功能需求**：

- 页面加载时间不超过2秒
- 弹窗打开动画流畅，无卡顿
- 适配主流屏幕尺寸（1920×1080、1366×768、1280×960等）
- 移动端适配（屏幕宽度＜992px时切换为上下布局）

## 十一、开发注意事项

### 11.1 关键点

1. **数据加载顺序**：先加载基础信息，再加载评论列表
2. **状态管理**：使用 Context 共享弹窗状态
3. **错误处理**：网络请求失败时显示错误提示
4. **空数据处理**：无数据时显示占位内容
5. **安全性**：用户输入需要进行 XSS 过滤

1. 使用 React.memo 优化组件渲染

3. 使用 useCallback 缓存回调函数
4. 图片懒加载减少首屏加载时间

1. 加载状态显示骨架屏
