# LiuYaoReault 组件修改计划书

## 1. 项目信息

- **文件路径**：`frontend/src/components/LiuYao/LiuYaoReault/LiuYaoReault.jsx`
- **文件类型**：React 组件（页面级）
- **修改日期**：2026-03-14
- **修改目标**：集成 FourPillarsDisplay 组件的显示控制功能，增强六爻排盘结果页面的显示控制能力

## 3. 修改方案

### 3.1 核心功能修改

#### 3.1.1 添加四柱显示控制状态
- **修改内容**：添加四柱显示配置状态
- **实现方式**：
  ```jsx
  // 四柱显示控制状态
  const [fourPillarsConfig, setFourPillarsConfig] = useState({
    rows: { year: true, month: true, day: true, hour: true },
    columns: { label: true, gan: true, zhi: true, vacancy: true },
    elements: {
      year: { label: true, gan: true, zhi: true, vacancy: true },
      month: { label: true, gan: true, zhi: true, vacancy: true },
      day: { label: true, gan: true, zhi: true, vacancy: true },
      hour: { label: true, gan: true, zhi: true, vacancy: true }
    }
  });
  ```

#### 3.1.2 增强 DisplayControl 组件
- **修改内容**：扩展 DisplayControl 组件，添加四柱显示控制按钮
- **实现方式**：
  ```jsx
  <DisplayControl
    activeButtons={activeButtons}
    onButtonClick={handleDisplayControlClick}
    fourPillarsConfig={fourPillarsConfig}
    onFourPillarsConfigChange={setFourPillarsConfig}
  />
  ```

#### 3.1.3 传递配置给 FourPillarsDisplay 组件
- **修改内容**：将显示配置传递给 FourPillarsDisplay 组件
- **实现方式**：
  ```jsx
  <FourPillarsDisplay
    ganzhiInfo={divinationData.calendar_info?.ganzhi_info || {}}
    isColorMode={activeButtons.includes('colorChange')}
    displayConfig={fourPillarsConfig}
  />
  ```

### 3.2 响应式优化

#### 3.2.1 添加响应式布局调整
- **修改内容**：根据屏幕尺寸自动调整四柱显示配置
- **实现方式**：
  ```jsx
  useEffect(() => {
    const handleResize = () => {
      const width = window.innerWidth;
      
      // 调整四柱显示配置
      setFourPillarsConfig(prev => ({
        ...prev,
        columns: {
          ...prev.columns,
          label: width >= 768,
          vacancy: width >= 480
        }
      }));
    };
    
    window.addEventListener('resize', handleResize);
    handleResize(); // 初始化
    
    return () => window.removeEventListener('resize', handleResize);
  }, []);
  ```

### 3.3 六爻网格显示控制增强

#### 3.3.1 扩展六爻显示控制
- **修改内容**：添加对 LiuYaoGridDisplay 组件的显示控制
- **实现方式**：
  ```jsx
  // 六爻显示控制状态
  const [liuYaoConfig, setLiuYaoConfig] = useState({
    showAccessory: true,

    showShiYing: true,
    showFuShen: true
  });
  
  // 传递给 LiuYaoGridDisplay

  />
  ```

### 3.4 状态管理优化

#### 3.4.1 统一状态管理
- **修改内容**：使用 useReducer 统一管理显示控制状态
- **实现方式**：
  ```jsx
  const displayControlReducer = (state, action) => {
    switch (action.type) {
      case 'TOGGLE_BUTTON':
        return {
          ...state,
          activeButtons: state.activeButtons.includes(action.payload)
            ? state.activeButtons.filter(id => id !== action.payload)
            : [...state.activeButtons, action.payload]
        };
      case 'UPDATE_FOUR_PILLARS_CONFIG':
        return {
          ...state,
          fourPillarsConfig: action.payload
        };
      case 'UPDATE_LIU_YAO_CONFIG':
        return {
          ...state,
          liuYaoConfig: action.payload
        };
      default:
        return state;
    }
  };
  
  const [displayState, dispatch] = useReducer(displayControlReducer, {
    activeButtons: ['liuqin', 'fuchen', 'yinyang', 'colorChange'],
    fourPillarsConfig: defaultFourPillarsConfig,
    liuYaoConfig: defaultLiuYaoConfig
  });
  ```

## 4. 性能优化

### 4.1 状态记忆化
- **修改内容**：使用 useMemo 缓存计算结果
- **实现方式**：
  ```jsx
  const displayStateMemoized = React.useMemo(() => ({
    ...displayState
  }), [displayState]);
  ```

### 4.2 事件处理优化
- **修改内容**：使用 useCallback 缓存事件处理函数
- **实现方式**：
  ```jsx
  const handleDisplayControlClick = React.useCallback((buttonId) => {
    dispatch({ type: 'TOGGLE_BUTTON', payload: buttonId });
  }, []);
  
  const handleFourPillarsConfigChange = React.useCallback((config) => {
    dispatch({ type: 'UPDATE_FOUR_PILLARS_CONFIG', payload: config });
  }, []);
  ```

### 4.3 响应式优化

- **实现方式**：

  ```jsx
    // 响应式逻辑
  }, 100), []);
  ```
