## 六、常见问题解决

### 6.1 前端问题

**问题 1：npm install 失败**
- **原因**：网络问题或依赖冲突
- **解决方法**：
  - 检查网络连接
  - 清除 npm 缓存：`npm cache clean --force`
  - 使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`

**问题 2：端口 3001 被占用**
- **原因**：其他进程占用了 3001 端口
- **解决方法**：
  - 修改 package.json 中的端口设置
  - 或终止占用端口的进程

### 6.2 后端问题

**问题 1：虚拟环境激活失败**
- **原因**：路径错误或权限问题
- **解决方法**：
  - 检查路径是否正确
  - 以管理员身份运行命令提示符

**问题 2：模块导入错误**
- **原因**：依赖未安装或路径问题
- **解决方法**：
  - 确保虚拟环境已激活
  - 重新安装依赖
  - 检查导入路径是否正确

**问题 3：数据库迁移失败**
- **原因**：数据库文件权限或结构问题
- **解决方法**：
  - 检查数据库文件权限
  - 确保备份数据库后再执行迁移
  - 查看迁移脚本错误信息

### 6.3 前后端交互问题

**问题 1：API 调用失败**
- **原因**：后端服务器未启动或跨域问题
- **解决方法**：
  - 确保后端服务器已启动
  - 检查 API 地址配置是否正确
  - 确保后端已配置 CORS

**问题 2：数据返回格式错误**
- **原因**：前后端数据结构不一致
- **解决方法**：
  - 检查 API 响应格式
  - 确保前端数据处理逻辑正确
