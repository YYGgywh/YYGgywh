```
       ^                         ^                   | pan_id (FK) |
       |                         |                   | ...        |
       | 1:N                     | 1:N              +------------+
       |                         |
+------------+                +------------+
| pan_collect|----------------+            |
+------------+                |            |

| user_id (FK)|               |            |
| pan_id (FK) |               |            |
| ...        |               |            |
+------------+               +------------+
```

### 5.2 关系说明

1. **user与pan_record**：
   - 一对多关系：一个用户可以创建多个排盘记录
   - 外键：`pan_record.user_id` 引用 `user.id`

2. **user与pan_like**：
   - 一对多关系：一个用户可以点赞多个排盘记录
   - 外键：`pan_like.user_id` 引用 `user.id`

3. **pan_record与pan_like**：
   - 一对多关系：一个排盘记录可以被多个用户点赞
   - 外键：`pan_like.pan_id` 引用 `pan_record.id`

4. **user与pan_collect**：
   - 一对多关系：一个用户可以收藏多个排盘记录
   - 外键：`pan_collect.user_id` 引用 `user.id`

5. **pan_record与pan_collect**：
   - 一对多关系：一个排盘记录可以被多个用户收藏
   - 外键：`pan_collect.pan_id` 引用 `pan_record.id`
