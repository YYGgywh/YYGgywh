#### 任务1.1：用户注册接口敏感词集成

| 项目 | 详情 |
|------|------|
| **任务描述** | 在用户注册接口中添加昵称和登录名的敏感词检测 |
| **涉及文件** | [backend/app/api/user.py](file:///f:/Project/YYG_paipan_Project/backend/app/api/user.py#L129-L228) |
| **实施步骤** | 1. 在register()函数中，登录名生成后添加敏感词检测<br>2. 检测到禁止级敏感词时拒绝注册<br>3. 返回明确的错误提示信息 |
| **预期完成时间** | 1小时 |
| **验收标准** | 注册包含敏感词的昵称/登录名时被拒绝 |

**实施代码示例**：
```python
# 在注册逻辑中添加
from app.utils.sensitive_word_filter import create_filter

# 检查登录名敏感词
filter = create_filter(db)
is_safe, found_words = filter.check(login_name)

if not is_safe:
    forbidden_words = [w for w in found_words if w['level'] == 2]
    if forbidden_words:
        word_list = ", ".join([w['word'] for w in forbidden_words])
        raise HTTPException(
            status_code=400, 
            detail=f"登录名包含禁止使用的敏感词：{word_list}，请修改后重试"
        )
```
